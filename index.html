FontOzy

Collecting my favorites from Google Fonts and Font Squirrel。

This project is based on Chad Mazzola's Beautiful Web Type, a website that showcases some of the author's favorite fonts from Google Fonts, a content delivery network (CDN) for a collection of free fonts. Inspired by his work, I wanted to present my own collection of free and well designed fonts. They can be found at Google Fonts or Font Squirrel, which provides an even larger selection of free fonts. Unlike Google Fonts, however, Font Squirrel unfortunately does not provide a CDN service for their fonts, so users must download the fonts they want and include them via CSS3's @font-face rule.

What are monospaced fonts?

"Monospaced fonts were widely used in early computers and computer terminals, which often had extremely limited graphical capabilities. Hardware implementation was simplified by using a text mode where the screen layout was addressed as a regular grid of tiles, each of which could be set to display a character by indexing into the hardware's character map. Some systems allowed colored text to be displayed by varying the foreground and background color for each tile. Other effects included reverse video and blinking text. Nevertheless, these early systems were typically limited to a single console font."

– Wikipedia contributors, "Monospaced font," Wikipedia, The Free Encyclopedia
